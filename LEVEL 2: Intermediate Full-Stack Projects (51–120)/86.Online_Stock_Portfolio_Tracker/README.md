# Project 86: 📈 StockTrack

## Description
📈 StockTrack - A Full Stack Web Application for FSD Course (Level 2)

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express.js
- **Database**: In-Memory (can be extended to MongoDB/MySQL)

## Setup Instructions

### Prerequisites
- Node.js (v14 or above)
- npm

### Steps
1. Navigate to project folder:
   ```
   cd 86_Online_Stock_Portfolio_Tracker
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
4. Open browser at: **http://localhost:3000**

## Features
- CRUD operations for all entities
- RESTful API with Express.js
- Responsive UI with dark theme
- Dashboard with live statistics

## API Endpoints
- `GET/POST /api/stocks` — Manage Stocks
- `DELETE /api/stocks/:id` — Delete Stock
- `GET/POST /api/portfolio` — Manage Portfolio
- `DELETE /api/portfolio/:id` — Delete Portfolio
- `GET/POST /api/watchlist` — Manage Watchlist
- `DELETE /api/watchlist/:id` — Delete Watchlist
