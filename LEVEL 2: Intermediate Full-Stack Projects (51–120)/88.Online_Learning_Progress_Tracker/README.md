# Project 88: 📊 LearnTrack

## Description
📊 LearnTrack - A Full Stack Web Application for FSD Course (Level 2)

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express.js
- **Database**: In-Memory (can be extended to MongoDB/MySQL)

## Setup Instructions

### Prerequisites
- Node.js (v14 or above)
- npm

### Steps
1. Navigate to project folder:
   ```
   cd 88_Online_Learning_Progress_Tracker
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
4. Open browser at: **http://localhost:3000**

## Features
- CRUD operations for all entities
- RESTful API with Express.js
- Responsive UI with dark theme
- Dashboard with live statistics

## API Endpoints
- `GET/POST /api/courses` — Manage Courses
- `DELETE /api/courses/:id` — Delete Course
- `GET/POST /api/students` — Manage Students
- `DELETE /api/students/:id` — Delete Student
- `GET/POST /api/progress` — Manage Progress
- `DELETE /api/progress/:id` — Delete Progre
