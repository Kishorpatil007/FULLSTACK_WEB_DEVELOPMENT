# Project 96: 🍳 RecipeShare

## Description
🍳 RecipeShare - A Full Stack Web Application for FSD Course (Level 2)

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express.js
- **Database**: In-Memory (can be extended to MongoDB/MySQL)

## Setup Instructions

### Prerequisites
- Node.js (v14 or above)
- npm

### Steps
1. Navigate to project folder:
   ```
   cd 96_Online_Recipe_Sharing_Platform
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
4. Open browser at: **http://localhost:3000**

## Features
- CRUD operations for all entities
- RESTful API with Express.js
- Responsive UI with dark theme
- Dashboard with live statistics

## API Endpoints
- `GET/POST /api/recipes` — Manage Recipes
- `DELETE /api/recipes/:id` — Delete Recipe
- `GET/POST /api/chefs` — Manage Chefs
- `DELETE /api/chefs/:id` — Delete Chef
- `GET/POST /api/categories` — Manage Categories
- `DELETE /api/categories/:id` — Delete Categorie
