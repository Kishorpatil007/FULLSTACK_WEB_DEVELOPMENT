# Project 98: 🔨 AuctionHub

## Description
🔨 AuctionHub - A Full Stack Web Application for FSD Course (Level 2)

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express.js
- **Database**: In-Memory (can be extended to MongoDB/MySQL)

## Setup Instructions

### Prerequisites
- Node.js (v14 or above)
- npm

### Steps
1. Navigate to project folder:
   ```
   cd 98_Online_Auction_System
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
4. Open browser at: **http://localhost:3000**

## Features
- CRUD operations for all entities
- RESTful API with Express.js
- Responsive UI with dark theme
- Dashboard with live statistics

## API Endpoints
- `GET/POST /api/items` — Manage Items
- `DELETE /api/items/:id` — Delete Item
- `GET/POST /api/bids` — Manage Bids
- `DELETE /api/bids/:id` — Delete Bid
