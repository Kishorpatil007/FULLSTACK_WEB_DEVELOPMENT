# Project 94: 💪 FitTrack

## Description
💪 FitTrack - A Full Stack Web Application for FSD Course (Level 2)

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express.js
- **Database**: In-Memory (can be extended to MongoDB/MySQL)

## Setup Instructions

### Prerequisites
- Node.js (v14 or above)
- npm

### Steps
1. Navigate to project folder:
   ```
   cd 94_Online_Fitness_Tracking_Platform
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
4. Open browser at: **http://localhost:3000**

## Features
- CRUD operations for all entities
- RESTful API with Express.js
- Responsive UI with dark theme
- Dashboard with live statistics

## API Endpoints
- `GET/POST /api/members` — Manage Members
- `DELETE /api/members/:id` — Delete Member
- `GET/POST /api/workouts` — Manage Workouts
- `DELETE /api/workouts/:id` — Delete Workout
- `GET/POST /api/plans` — Manage Plans
- `DELETE /api/plans/:id` — Delete Plan
