# Project 92: 🏫 ClassPortal

## Description
🏫 ClassPortal - A Full Stack Web Application for FSD Course (Level 2)

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express.js
- **Database**: In-Memory (can be extended to MongoDB/MySQL)

## Setup Instructions

### Prerequisites
- Node.js (v14 or above)
- npm

### Steps
1. Navigate to project folder:
   ```
   cd 92_Online_Classroom_Portal
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
4. Open browser at: **http://localhost:3000**

## Features
- CRUD operations for all entities
- RESTful API with Express.js
- Responsive UI with dark theme
- Dashboard with live statistics

## API Endpoints
- `GET/POST /api/classes` — Manage Classes
- `DELETE /api/classes/:id` — Delete Classe
- `GET/POST /api/students` — Manage Students
- `DELETE /api/students/:id` — Delete Student
- `GET/POST /api/assignments` — Manage Assignments
- `DELETE /api/assignments/:id` — Delete Assignment
