# Project 100: 📊 FeedbackIQ

## Description
📊 FeedbackIQ - A Full Stack Web Application for FSD Course (Level 2)

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express.js
- **Database**: In-Memory (can be extended to MongoDB/MySQL)

## Setup Instructions

### Prerequisites
- Node.js (v14 or above)
- npm

### Steps
1. Navigate to project folder:
   ```
   cd 100_Online_Feedback_Analytics_Dashboard
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
4. Open browser at: **http://localhost:3000**

## Features
- CRUD operations for all entities
- RESTful API with Express.js
- Responsive UI with dark theme
- Dashboard with live statistics

## API Endpoints
- `GET/POST /api/surveys` — Manage Surveys
- `DELETE /api/surveys/:id` — Delete Survey
- `GET/POST /api/responses` — Manage Responses
- `DELETE /api/responses/:id` — Delete Response
