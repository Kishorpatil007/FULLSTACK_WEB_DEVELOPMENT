# Project 93: 🏢 RoomBook

## Description
🏢 RoomBook - A Full Stack Web Application for FSD Course (Level 2)

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express.js
- **Database**: In-Memory (can be extended to MongoDB/MySQL)

## Setup Instructions

### Prerequisites
- Node.js (v14 or above)
- npm

### Steps
1. Navigate to project folder:
   ```
   cd 93_Online_Meeting_Room_Booking
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
4. Open browser at: **http://localhost:3000**

## Features
- CRUD operations for all entities
- RESTful API with Express.js
- Responsive UI with dark theme
- Dashboard with live statistics

## API Endpoints
- `GET/POST /api/rooms` — Manage Rooms
- `DELETE /api/rooms/:id` — Delete Room
- `GET/POST /api/bookings` — Manage Bookings
- `DELETE /api/bookings/:id` — Delete Booking
