# Project 81: Online Resume Screening System

## Description
📄 ResumeScreen — An AI-assisted resume screening platform for recruiters.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express.js
- **Database**: In-Memory

## Setup
1. `npm install`
2. `npm start`
3. Open http://localhost:3000

## Features
- Post job openings with required skills and experience
- Upload candidate resumes
- Auto-screen resumes against job requirements with scoring
- View shortlisted vs rejected candidates

## API Endpoints
- `GET/POST /api/jobs` — Manage job postings
- `GET/POST /api/resumes` — Manage resumes
- `GET/POST /api/screenings` — Run screenings
