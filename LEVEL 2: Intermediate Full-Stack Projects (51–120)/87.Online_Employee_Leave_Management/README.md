# Project 87: 🏖 LeaveManager

## Description
🏖 LeaveManager - A Full Stack Web Application for FSD Course (Level 2)

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express.js
- **Database**: In-Memory (can be extended to MongoDB/MySQL)

## Setup Instructions

### Prerequisites
- Node.js (v14 or above)
- npm

### Steps
1. Navigate to project folder:
   ```
   cd 87_Online_Employee_Leave_Management
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
4. Open browser at: **http://localhost:3000**

## Features
- CRUD operations for all entities
- RESTful API with Express.js
- Responsive UI with dark theme
- Dashboard with live statistics

## API Endpoints
- `GET/POST /api/employees` — Manage Employees
- `DELETE /api/employees/:id` — Delete Employee
- `GET/POST /api/leaves` — Manage Leave Requests
- `DELETE /api/leaves/:id` — Delete Leave Request
- `GET/POST /api/holidays` — Manage Holidays
- `DELETE /api/holidays/:id` — Delete Holiday
